CDP RELEASE 8
=============+
This folder contains the latest (March 2023) CDP program set for PC,
including the new CDP Release 8 programs.

It includes programs that are needed for Soundloom (only) and some extras:
mf2t.exe and t2mf.exe (utilities for TV) and robocopy (for Soundshaper).

Also included are ness-brass.exe and assocated files (for Soundloom).
