CDP CHARTS AND GLOSSARIES

The following glossaries and charts are available:

SPIRAL BOUND DESK REFERENCE (in pdf format)
Title Page
Contents
Desk Reference
Printable list of all CDP functions (up to Release 7), classified into function groups and CDP packages.
Modes are also listed, using SoundLoom labels

CDP FILES AND CODES
The definitive guide to the various input codes, text files and binary files used in CDP processes, with examples.
TECHNICAL GLOSSARY
An explanation of computer audio terms used in CDP programs and documentation.
MUSICAL GLOSSARY
An explanation of key musical terms needed for manipulating sound.
PITCH CHART
Convert pitch and MIDI-pitch to Frequency. (Pdf version)
DURATIONS CHART
Convert common rhythmic values into durations for Texture or Sequence programs. (In pdf format)
GAIN to dB CHART
Convert gain increase or reduction to equivalent decibel values. (Pdf version)
TRANSPOSITION CHART
Conversion between semitones and transposition ratios: up and down. (Pdf version)
TRANSPOSITION & FREQUENCY-SHIFT CHART
Inputs and Outputs for Spectral transposition & frequency-shifting. (Pdf version)
CDP FUNCTION ABBREVIATIONS
Suggested abbreviations for key CDP functions in filenames. (Pdf version)

TEXTURE: NOTE DATA FILES
Chart of Note data file requirements for the different Texture programs. (Pdf version)
TEXTURE: 2 REFERENCE CHARTS
Two further charts relating to Texture Note data files. (Pdf version)

AUDIBLE DESIGN TO CDP CHART
A chart linking the processes described in Trevor Wishart's 1994 book Audible Design to the equivalent CDP processes. (Pdf version)
