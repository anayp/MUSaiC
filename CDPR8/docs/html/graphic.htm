CDP GRAPHIC PROGRAMS

Graphical User Interfaces

Various front-ends give access to the CDP system:

Sound Loom

Supports the full set of CDP processes, plus some extras. Available for PC and Mac.
  • User's Guide to the Sound Loom & CDP (in pdf format)
  • Sound Loom Reference Manual
Author: Trevor Wishart
  • Sound Loom Basic Guide (in pdf format)
Author: Archer Endrich

Soundshaper

Supports the full set of CDP processes. Currently available for PC only.
Built around a multi-process patch grid and supports mono/stereo/multi-channel input.
  • About Soundshaper (pdf version)
  • Soundshaper Basic Operation (pdf format)
Author: Robert Fraser

SoundThread

A new cross-platform interface, currently in Beta. Supports around 100 CDP processes.
Node-based, with processes patched together in series or parallel and then run as a 'Thread'.
  • Description on Github, with further details.
  • First of three introductory videos on YouTube.
Author: Jonathon Higgins

Renoise CDP Interface

A new interface integrated into the RENOISE DAW, supporting most CDP processes.
The interface is multi-platform and is currently the only working CDP interface for LINUX.
  • CDP Interface description
  • Linux CDP Tool Guide (Dave Phillips)
Authors: Aftab Hussain (Lua programming) and Richard Jonas (CDP definition files)

Other CDP Supplied Programs

GrainMill

Graphic Editor for granular reconstruction of soundfiles (c.f. MODIFY BRASSAGE).
  • About CDP GrainMill
    CDP Information sheet about GrainMill (.pdf)
  • GrainMill Reference
  • GrainMill Tutorials
Author: Richard Dobson

BRKEDIT

Breakpoint graph editor which can display more than one file at a time. You can create exponential and logarithmic curves as well as linear shapes.
  • Brkedit Manual
Author: Richard Dobson

ViewSF

A soundfile viewer with the ability to zoom in to sample level. VIEWSF can also display sonograms of frequency analysis files. (Currently undocumented.)

Author: Richard Dobson

Tabula Vigilans

Tabula Vigilans ('T-V') is an advanced (non-graphic) scripting language, aimed at algorithmic composition. As a score processor, it can define and replay anything from completely conventional melodic and harmonic music to envelope-shaped multi-event textures with random features. T.V. can operate as a real-time MIDI instrument to play pre-programmed or instantly generated music under mouse or keyboard control. It can also call and run any of the CDP programs, in command-line form. TV (version 1.5) is included in the CDP download, but for a full install, download T.V. zip:

  • T.V. zip file: tutorials / manual / supporting files etc.
  • T.V. Information Sheet
  • Tabula Vigilans Manual
Author: Richard Orton

Also available:

ProcessPack

A graphic program suite complementary to CDP, with an emphasis on algorithmic procedures in micro-time.
  • Process Pack Description
Authors: Richard Orton & Archer Endrich (available separately)

AL/Erwin

AL is an algorithmic compositional environment for the creation and manipulation of sonic events and their organization into a structured musical work. ERWIN is a plug-in for AL applying the solutions of Schroedinger's equation to the creation of granular clouds.
  • AL/Erwin: details and download
  • AL/Erwin Information Sheet
Author: Rajmil Fischman

Clouds

Clouds implements real-time asynchronous granular synthesis, with four audio buffers, comprehensive controls and up to 8-channel output.
  • Clouds: details and download
Author: Rajmil Fischman
