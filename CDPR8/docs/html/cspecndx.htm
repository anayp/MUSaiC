CDP Spectral Processing Function Groups

The CDP Spectral processes operate mainly on Frequency Analysis files (.ana), derived from sound files using the FFT function PVOC ANAL. An alternative supported format is PVOC-EX (.pvx). Other processes manipulate pitch data extracted from the time-varying spectrum (see REPITCH). See the INTRODUCTION for more detail on the nature of processing in the spectral domain.

CLICK ON THE GROUP NAME TO THE LEFT TO ACCESS THE CDP REFERENCE DOCUMENTATION ON THE FUNCTION GROUP.
CLICK ON THE GROUP NAME BELOW TO SEE A LIST OF FUNCTIONS IN THE GROUP.

BLUR (Time) – various means to diffuse the clarity of the original sound
COMBINE (Spectra) – combine analysis data from two or more files
FOCUS – isolate and hold particular data
FORMANTS – formant data, more generally, the spectral envelope
HILITE (Amplitude) – pick out and emphasise various aspects of analysis data
MORPH – create smooth transitions between sounds, using amplitude and frequency analysis data
ONEFORM – operations with single formants
PITCH (& Harmony/Freq) – operations based on partials
PITCHINFO (Pitch Data Info) – access information about partials or extracted pitch traces
PVOC – FFT analysis and resynthesis
REPITCH (Pitch Data) – extract and alter pitch-related data
SPEC (Gain/Edit) – basic utilities in the spectral domain
SPECFNU – Modify spectral shape in relation to formant peaks
SPECNU – various functions, mainly to clean up analysis files
SPECINFO – information about spectral data
STRANGE – some unusual and unpredictable effects
STRETCH – stretch time or frequency data
