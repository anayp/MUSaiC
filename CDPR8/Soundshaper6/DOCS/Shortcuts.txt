SUMMARY OF MAIN PAGE SHORTCUTS:
-------------------------------
Function keys
-------------
F keys and Shift/Ctrl/Alt combinations are mostly 
associated with a particular function type:

1 - Help, 2 - View, 3 - Markers, 4 - Patch, 5 - Settings, 6 - Graph-Edit
7 - Pool, 8 - Process selection, 9 - Presets, 10-12 Player

Alt F6-F12 are associated with External applications:
6 - MIDI-Seq.	7 - Pitch Calc   9 -Calc  10-12 External Progs 
  
Loads are CTRL+ALT; Saves are SHIFT+ALT

Shift+F	- Soundshaper Options

Letters
-------
CTRL+		File operations and top-level controls
ALT+		Main MENUS + some top-level controls
Shift+Ctrl+	SUBMENUS- Soundfile
Ctrl+Alt+	SUBMENUS - Spectral
Shift+Alt+	Files/Processes etc Panels

Numbers
-------
CTRL+		Select column of active Patch row
Shift+Ctrl+	Select Patch row
ALT+		Process-selection tables; Generate Data
Ctrl+Alt+	Info
Shift+Alt+	 -- to be assigned --
